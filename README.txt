Так як дана програма використовує MS SQL Server, то для комфортного та повного використання програми рекомендується згенерувати скрипт бази (script.sql).

Для входу як адмін:
	login: admin
	pass: 123
Для входу як користувач:
	login: user
	pass: 123

Деякі зображення підгружаються з інтернету, тому бажано мати доступ до інтернету.
